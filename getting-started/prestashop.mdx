PrestaShop is a free, open-source e-commerce platform that allows retailers to create, manage, and grow their online stores. Edgee can be seamlessly placed in front of PrestaShop to simplify and optimize the integration process.  


### Overview: Using Edgee with Prestashop

Integrating Edgee with Prestashop is straightforward. The setup involves three main steps:

1. Download the Edgee Prestashop Module.  
2. Install the module in Prestashop.  
3. Configure Edgee in front of Prestashop [here](/getting-started/project-settings).  

<Note>Note: The module has been tested with Prestashop versions 8.1.x.</Note>


### Step-by-Step Guide: Setting up Edgee with Prestashop

1. Download the module: [Download ZIP](/downloads/prestashop_remoteipforward.zip).  

2. Log in to your Prestashop **Back Office**.
